## GitHub Search  101

### 1. Scope

```
in:file
in:path
in:file,path
```

Example

```
octocat in:file
octocat in:path
octocat in:file,path
```

### 2. Language

```
language:LANGUAGE
```

Example

```
element language:xml
display language:scss
```

### 3. File Name

```
filename:FILENAME
```

Example

```
filename:.vimrc
filename:appsettings.json
filename:tsconfig.json
```

### 4. File Extension

```
extension:EXTENSION
```

Example

```
icon extension:css
```

### 5. User / Organization / Repository

```
user:USERNAME
org:ORGNAME
repo:USERNAME/REPOSITORY
```

Example

```
user:defunkt extension:rb
org:github extension:js
repo:mozilla/shumway extension:as
```
