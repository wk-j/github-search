## GitHub Search  101

### Scope

- in:file
- in:path
- in:file,path

**Exapmle**

```
octocat in:file
octocat in:path
octocat in:file,path
```

### Language

- language:LANGUAGE

**Example**

```
element language:xml
display language:scss
```

### File Name

- filename:FILENAME

**Example**

```
filename:.vimrc
filename:appsettings.json
filename:tsconfig.json
```

### File Extension

- extension:EXTENSION

**Example**

```
icon extension:css
```

### User / Organization

- user:USERNAME
- org:ORGNAME
- repo:USERNAME/REPOSITORY

**Example**

```
user:defunkt extension:rb
org:github extension:js
repo:mozilla/shumway extension:as
```
